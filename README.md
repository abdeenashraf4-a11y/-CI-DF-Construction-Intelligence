# Construction Intelligence (CI-DF)

## Decision-Centric Framework for Engineering Project Contracting

This repository contains the official SSRN preprint of the Construction Intelligence Decision Framework (CI-DF), an integrated decision-centric system designed to transform engineering project contracting into an adaptive intelligent ecosystem.

## Key Concept

CI-DF introduces Construction Decision Theory (CDT) and integrates multiple layers:

- SCIM (System Construction Intelligence Model)
- CDS (Construction Decision System)
- PLDS (Project Lifecycle Decision System)
- GACM (Global Adaptation Model)
- SECIM (Self-Evolving Construction Intelligence Model)
- DCIM (Dynamic Cost Intelligence Model)
- CPOM (Construction Performance Optimization Model)

## Core Contribution

The framework establishes a new principle:

> Construction system validity is defined by contextually correct outputs, not uniform results across environments.
